# Platformer_Lr
# 2D Platformer Runner with Hook and Reality Shift

## Описание
Проект представляет собой 2D платформер-раннер с вертикальным геймплеем, крюком и системой смены реальности.

Игроку необходимо подниматься по платформам, избегать ловушек и опережать движущуюся стену огня.

## Управление
- A / D — движение
- Space — прыжок
- S — провал с платформы
- ЛКМ — крюк
- E + ЛКМ — притягивание
- Q — смена реальности

## Основные механики
- Процедурная генерация уровней
- Физический крюк
- Ловушки: пилы, шипы, огонь
- Бонусы: скорость, прыжок, очки
- Два слоя реальности

## Запуск
Открыть файл index.html в браузере.
